🔄 Yandex Proxy API

✅ API работает

Это API-сервис для загрузки файлов в Vercel Blob, используемый чатом на GitHub Pages.

🔗 API endpoint: /api/generate-upload-url

📘 Использование:

GET запрос с параметрами:

  • filename — имя файла
  • filetype — MIME-тип файла

Пример:

https://yandex-proxy-murex.vercel.app/api/generate-upload-url?filename=image.jpg&filetype=image/jpeg
⚡ Этот сервис предназначен только для API-запросов.
Основной чат доступен на kolocuz.github.io
